# mini-proyecto
proyecto# Proyecto: Métodos de Derivación Numérica en Scala

## Objetivos
- Aplicar conocimientos sobre funciones de orden superior.
- Utilizar funciones como parámetros para calcular derivadas.
- Implementar manejo adecuado de excepciones.

## Métodos Implementados
1. **Diferencias Finitas Adelante**
2. **Diferencias Finitas Centradas**
3. **Diferencias Finitas Atrás**

Cada método incluye validación de parámetros y manejo de errores.

## Funciones Probadas
1. f(x) = x³ + 2x + 1  
2. f(x) = sin(x)  
3. f(x) = eˣ  

Todas evaluadas con **h = 0.0001**.

## Cálculo del Error
Se calcula el error absoluto para cada método como:

```
Error = |valor_real − valor_aproximado|
```
